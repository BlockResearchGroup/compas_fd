ShellArtist

class compas_fofin.rhino.ShellArtist(mesh, layer=None)

Bases: compas_rhino.artists.MeshArtist

The ShellArtist provides functionality for visualisation of shell components in Rhino.

Examples

artist = ShellArtist(shell, layer="Shell")
artist.clear_layer()
artist.draw_vertices()
artist.draw_edges()
artist.draw_faces()
artist.draw_forces()
artist.draw_reactions()
artist.redraw()
