CablenetArtist

class compas_fofin.rhino.CablenetArtist(cablenet, layer=None)

Bases: compas_rhino.artists.MeshArtist

The CablenetArtist provides functionality for visualisation of cablenet components in Rhino.

Examples

artist = CablenetArtist(cablenet, layer="Cablenet")
artist.clear_layer()
artist.draw_vertices()
artist.draw_edges()
artist.draw_faces()
artist.draw_forces()
artist.draw_reactions()
artist.redraw()
